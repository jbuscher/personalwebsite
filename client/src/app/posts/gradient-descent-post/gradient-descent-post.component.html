What is a Gradient Descent?

The concept of gradient descent is actually quite simple, it's a general algorithm for deterimining local mimima for any differentiable function. In other words, for a smooth function, running gradient descent on that function will find one of the valleys in that function, but not necessarily the lowest valley.
