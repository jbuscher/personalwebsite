I've spent a bit of time this year making sure I don't fall to far behind on this machine learning train. Unfortuntely, I'm so far removed from my initial lessons in college, it means I'm starting pretty much. from scratch. This lesson by 3Blue1Brown inspired me to try my hand at the basic "hello, world" project of neural networks. Handwriting recognition.

A weekend of work later, I managed to build a basic model with ~90% accuracy on the training data. The accuracy seems to be a little worse testing it with friends, but that's to be expected due to the differences from the training set and my janky UI for handwriting in the demo project.
